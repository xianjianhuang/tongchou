## 🔥 你不装统筹，你的 Agent 就是个高级搜索引擎

**你买了个 iPhone，但只能打电话——这就是没有统筹的 AI Agent。**

| 没有统筹 | **有了统筹** |
|---------|-------------|
| 你跟 Agent 说「帮我做个事」，它问你「你想用什么工具」 | **「安排。」** — 统筹自动判断用啥、怎么用、按什么顺序 |
| 要组合 3 个功能 → 你得手动串流程，写 prompt | **「一条龙。」** — 统筹自动并联/串联，你等结果就行 |
| Agent 不知道你装了什么 skill → 你告诉它 | **「有什么技能可以用？」** — 统筹扫描全库，按需求推荐 |
| 失败了一个步骤 → 整个流程崩了 | **「换个方案。」** — 统筹自动换备选 skill，从不空手而归 |

**一句话：别人家的 Agent 是工具人，你的 Agent 是 CEO。你把目标告诉统筹，剩下的它搞定。**

## 🔧 失败？不存在的。统筹有 5 套备案

| 翻车场景 | 统筹怎么救 | 你会看到 |
|----------|-----------|---------|
| 匹配到的 skill 不存在 | 自动重新扫描，换备选 | 任务继续，不受影响 |
| 子 agent 欠费 403 | 回退到主 agent 直接执行 | 结果照出，只是慢点 |
| 需求太模糊无法匹配 | 只问 1 个关键问题 | 「你想分析的是技术趋势还是商业模式？」 |
| skill 说「做不到」 | 换备选或用自身能力完成 | 换方案继续，不中断 |
| 某步卡住超时 | 跳过该步，继续后续 | 报告里标注「XX步骤超时已跳过」 |

**统筹的原则：绝不空手而归。哪怕所有技能都挂了，统筹用自己的能力给你一个交代。**
